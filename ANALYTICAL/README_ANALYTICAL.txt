Привіт, у даному файлі описано інструкцію по експлуатації другого тестового завдання хакатону INT20H.
Основний код програми прописано у файлі ltv.py.

Вимоги: 
- наявність версії Python 3.x;
- наявність бібліотеки pandas;
- наявність вхідніх даних для подальших обчислень, 
  а саме: data_analytics.csv разом з ltv.py в одній папці.
  
Щоб перевірити результати досліджень, треба запустити файл ltv.py:
- через редактор коду: натистуни ПКМ на файл ltv.py -> "Відкрити за допомогою..." -> обрати програму для запуску файлу(MVS, PyCharm тощо).
- за домогою консолі: перейдіть у папку з файлом -> натисніть ПКМ, зажавши Shift ->
  у контекстному меню обрати опцію типу "Відкрити командний рядок тут" -> ввести у
  командному рядку "python ltv.py". Після обчислення результатів у командному рядку
  буде показано результати обчислення у такому форматі:
  ...
  Total LTV among users: x
  Average LTV among users: y
  ...
  де x - загальний показник LTV за весь зазначений період від усіх користувачів разом,
  y - середній показник LTV серед усіх користувачів.
